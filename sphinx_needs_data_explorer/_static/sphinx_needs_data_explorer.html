Help Information

Keyboard shortcuts

  • visjs network keyboard shortcuts
    • arrow-up, arrow-down, arrow-left, arrow-right to move network on screen
    • +/- zoom-in/zoom out
  • shift arrow-up, shift arrow-down to navigate through sphinx_need item list
  • ENTER - to toggle network view between
    • selected sphinx_need item and all descendents (bottom down descent)
    • selected sphinx_need item and all descendents including all nodes reachable from descendents

Mouse events

  • hover over a node shows tooltip
  • a click at node shows detailed node information in window, including a link where complete information can be found
  • a click with pressed shift on node toggles collapse/expand node descendents

The following sphinx_needs attributes can be used in filters

  • id
  • type
  • status
  • title
  • docname
  • description

Examples of filtering

  • 'S_\' in id
  • 'arch' in docname
  • 'architecture/architecture-needs'==docname
  • status in ['open','closed']
  • (status=='open' && (type in ['req'])) || ('arch' in docname)
  • title ~ /^Level[ \t]+(?=one)/ #positive lookaheads
  • title ~ /^Level[ \t]+(?!one)/ #negative lookaheads
Filter matches '0' IDs
